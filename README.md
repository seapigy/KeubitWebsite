# KEUB Website

A premium, adventure-focused website for KEUB Smart Storage System.

## Brand Identity

KEUB is positioned as an outdoor freedom and adventure enablement platformâ€”not just a locker company. The brand message emphasizes that "the storage disappears, the adventure begins."

### Core Messages
- "One system. Infinite adventures."
- "Adventure starts when your pockets are empty."

### Visual Style
Inspired by premium outdoor brands like Patagonia, YETI, GoPro, Arc'teryx, and Airbnb Experiences. The design is:
- Rugged and premium
- Cinematic with large full-width imagery
- Clean, modern, and trustworthy
- Emotion and experience-driven

## Brand Colors

- **Primary Accent**: `#23DC7D` (used sparingly for CTAs, icons, highlights)
- **Dark Charcoal**: `#0F1A17` or `#111111`
- **Off-White**: `#F5F7F6`
- **Slate Gray**: `#6B7C76`
- **Earth Tone** (optional): `#C7B299` or `#9C8B6E`

## Typography

- **Headlines**: Playfair Display (bold, outdoor-inspired)
- **Body Text**: Inter (clean, modern sans-serif)
- High contrast between text and imagery

## Website Structure

1. **Hero Section** - Large outdoor imagery with main headline and CTA
2. **The Product** - Single product focus: KEUB Smart Storage System
3. **How It Works** - Simple 3-step process
4. **What KEUB Unlocks** - Experience showcase (rafting, skiing, beaches, festivals)
5. **One Platform, Many Environments** - Versatility demonstration
6. **For Operators & Municipalities** - Business benefits
7. **Call to Action** - Demo request section

## Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth scrolling navigation
- Mobile-friendly hamburger menu
- Contact form modal
- Intersection Observer animations
- Premium outdoor aesthetic
- Single product focus (no catalog)

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive functionality
- `README.md` - This file

## Usage

Simply open `index.html` in a web browser. No build process or dependencies required.

## Design Principles

- **Single Product Focus**: KEUB is one platform shown in different environments
- **Adventure First**: Storage is invisible, adventure is the focus
- **Premium Positioning**: Rugged, trustworthy, confident tone
- **Emotion-Driven**: Sell experience before technology

## Browser Support

Modern browsers (Chrome, Firefox, Safari, Edge) with CSS Grid and Flexbox support.
